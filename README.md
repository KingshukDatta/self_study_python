## üöÄ Getting Started

To explore the notes on your local machine, follow these steps:

### Prerequisites
Make sure you have Python installed. You will also need **Jupyter Notebook** or an IDE that supports it (like VS Code or PyCharm).

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/KingshukDatta/self_study_python.git](https://github.com/KingshukDatta/self_study_python.git)
   ```

2. **Navigate to the directory:**
  ```bash
  cd self_study_python

  ```


3. **Launch Jupyter Notebook:**
  ```bash
  jupyter notebook
  ```


4. Open `Python_Notes.ipynb` in your browser interface to view and run the code.
